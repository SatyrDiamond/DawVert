
# DawVert

## How to Use
```
python3 DawVert.py <input> <output type> <output>

input type: -it 
input file: -i 
output type: -ot 
output file: -o

e.g: 
    python3 DawVert.py -i song.txt -ot flp -o out.flp

    python3 DawVert.py -it fs_txt -i song.txt -ot flp -o out.flp

```
